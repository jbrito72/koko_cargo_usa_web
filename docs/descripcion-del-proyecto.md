Tenemos una aplicación que usa como base de datos postgresql 16.9 sobre Ubuntu 24.04, está aplicación se conecta principalmente a una base de datos llamada camaronera, donde se guarda información sobre la producción de una industria camaronera, además se conecta a otras bases de datos cada una de ellas representan una empresas para el manejo de inventarios todas las bases de datos están en el mismo cluster, la aplicación esta hecha en xojo 2023 r4. para web
